# Portfolio Generation Skill

Use this skill to transform raw project artifacts into a professional internship-ready showcase.

## Instructions

1. **Extract Project Metadata**:
    - Identify project Name, Tech Stack, Key Milestones, and Artifact Links.
2. **Generate JSON Database**:
    - Create a `data.json` following the schema observed in the Showcase project.
3. **Scaffold Dashboard**:
    - Use Vite + React to create a responsive, glassmorphic UI.
    - Componentize visualizations (Radar charts for skills, Timelines for projects).
4. **Linking**:
    - Ensure all dashboard elements hyperlink back to the actual source code and documentation.
